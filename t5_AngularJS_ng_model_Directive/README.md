## t1.html - Validate User Input

The ng-model directive adds/removes the following classes, according to the status of the form field:

* ng-empty
* ng-not-empty
* ng-touched
* ng-untouched
* ng-valid
* ng-invalid
* ng-dirty
* ng-pending
* ng-pristine

```css
input.ng-invalid {
    background-color: red;
}
```
